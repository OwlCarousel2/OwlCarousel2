Lazy Load

Overview

Add lazyLoad to plugin setup:

lazyLoad: true

LazyLoad HTML structure requires class="owl-lazy" and data-src="url_to_img" or/and data-src-retina="url_to_highres_img". If you set above settings not on <img> but on other DOM element then Owl will load an image into css inline background style.

Setup

$('.owl-carousel').owlCarousel({
    items:4,
    lazyLoad:true,
    loop:true,
    margin:10
});

HTML:

<div class="owl-carousel owl-theme">
